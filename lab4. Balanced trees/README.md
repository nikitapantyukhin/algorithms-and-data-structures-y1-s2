## Тема: Красно-чёрное дерево

### Описание лабы
Реализовать структуру данных «Красно-чёрное дерево» со строковыми ключами (длиной до 6 символов) и вещественными значениями. Программа должна обеспечивать автоматическую балансировку дерева при выполнении операций изменения структуры.

### Реализованный функционал
В работе реализованы 4 основные операции:
1.  **Добавление узла:** Вставка нового элемента по ключу с последующим восстановлением свойств красно-черного дерева
2.  **Удаление узла:** Поиск и удаление элемента по ключу с последующей балансировкой для сохранения черной высоты и других свойств дерева.
3.  **Поиск элемента:** Получение значения по заданному ключу.
4.  **Печать дерева:** Рекурсивный вывод структуры дерева в текстовый файл с отображением иерархии (отступы), связей (L/R) и обязательной маркировкой цвета каждого узла:
    *   `R` — Красный (Red)
    *   `B` — Чёрный (Black)

### Формат ввода
Программа считывает команды из файла `input.txt`. Каждая строка содержит номер операции и необходимые параметры:
*   `1 <key> <value>` — Добавить/обновить узел.
*   `2 <key>` — Удалить узел.
*   `3` — Вывести текущее состояние дерева.
*   `4 <key>` — Найти значение по ключу.
*   `0` — Завершить выполнение.

### Формат вывода
Результаты выполнения записываются в файл `output.txt`.

### Пример сборки и запуска
Программа написана на языке C. Для компиляции можно использовать `gcc`:
```bash
gcc main.c -o program
./program
```
