# Лабораторная работа №3: Арифметические выражения и деревья

**Вариант 17:**
Реализация правила перемножения степеней с одинаковыми основаниями:
` a^n * a^k  —>  a^(n+k) `

## Пример работы программы
**Ввод:**
` a^2 * a^k + 10 `

**Визуализация процесса:**
Программа находит поддерево умножения двух степеней `a^2` и `a^k`, проверяет, что основания `a` совпадают, и перестраивает дерево.

**Результат:**
` (a^(2.00+k)+10.00) `

   ```bash
   gcc main.c -o program
   ```
2. Запустить исполняемый файл:
   ```bash
   ./program
   ```
